Este desconto é válido até ao final do mês de março. Dá acesso a um desconto de 5% para qualquer valor de encomenda desde que o carrinho de compras inclua pelo menos um produto das seguintes categorias:

Arduino

Arduino kits

Arduino Shields

Isto permite que comprem um Raspberry com 5% de desconto mas só se comparem um produto da família Arduino. O sistema de códigos deste software não permite excluir produtos e torna as contas finais mais complicadas.

Este código é válido para pagamentos por Paypal ou transferência bancária, não tem mínimo de valor de compra, e só pode ser usado uma vez por pessoa.

A única restrição é que tem de conter pelo menos um dos produtos acima, mas desde que tenha um Arduino podem comprar um Berry.

O código BOT-OLYMPICS só permite utilizações individuais, 1 por utilizador.
